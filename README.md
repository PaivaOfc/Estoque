# 📦 Sistema de Gerenciamento de Estoque

Este projeto foi desenvolvido em **Java** com o objetivo de consolidar e aprimorar meus conhecimentos em lógica de programação e **Programação Orientada a Objetos (POO)**. O sistema simula o controle básico de um inventário, permitindo gerenciar produtos de forma eficiente via console.

---

## 🚀 Objetivo do Projeto
Desenvolvi este repositório como parte dos meus estudos práticos. O foco foi aplicar conceitos fundamentais da linguagem Java e boas práticas de desenvolvimento para criar uma solução funcional e organizada.

## ✨ Funcionalidades
- [x] **Cadastro de Produtos:** Registro de novos itens (Nome, SKU, Quantidade, Preço).
- [x] **Atualização de Estoque:** Controle de entrada e saída de mercadorias.
- [x] **Listagem Geral:** Visualização completa de todos os itens cadastrados.
- [x] **Remoção de Itens:** Exclusão de produtos do sistema.
- [x] **Busca Inteligente:** Localização de produtos específicos por nome ou código.

## 🛠️ Tecnologias e Conceitos Aplicados
* **Java JDK:** Linguagem base do projeto.
* **POO (Programação Orientada a Objetos):** Classes, objetos, encapsulamento e métodos.
* **Java Collections:** Uso de `ArrayList` para o gerenciamento dinâmico dos dados.
* **Tratamento de Exceções:** Garantia de que o programa não feche inesperadamente em caso de erros de entrada.
* **Entrada/Saída (I/O):** Interação com o usuário via terminal utilizando a classe `Scanner`.
